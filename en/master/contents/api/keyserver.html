KeyServer endpoints¶

Have a look at the README of the repo for a more up to date version of endpoint documentation.

In general all key.box endpoints are publicly available, as long as the request contents a valid userAccessToken.

POST /getKey¶

Requesting the encrypted key data for a given userAccessToken

Example request:

POST /getKey HTTP/1.1
Content-Type: application/json

Example response:

HTTP/1.1 200 OK
Content-Type: application/json

{
  "foo": 123
}
Form Parameters

  • userAccessToken – generated access token for one request

Status Codes

  • 200 OK – if token was valid

  • 404 Not Found – when parameters are missing or invalid
